lswicon (LEGO Star Wars Icon Generator)

This cool tool can generate icons from LEGO Star Wars for use in profile pictures.

It is open source and created with Godot Engine, the best game engine.


The newest version seen here is delayed due to a bug in Godot 4

The latest release can be downloaded here.
